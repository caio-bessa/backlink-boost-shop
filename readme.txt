A child theme of Astra for BacklinkBoost application.

== Description ==

BacklinkBoost is a WordPress child theme based on Astra for managing backlink campaigns and viewing SEO performance metrics.

== Installation ==

1. Upload the `backlinkboost` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Make sure the Astra theme is also installed

Required plugins:
- WooCommerce
- Font Awesome (or install using functions.php)

== Features ==

1. Custom app dashboard for managing backlink campaigns
2. Integration with Google Search Console and Google Analytics
3. Financial reporting
4. Link building campaign creation
5. Content strategy tools
6. Custom login and registration pages
7. WooCommerce integration with simplified checkout

== Setup ==

After activating the theme, the following pages will be automatically created:
- App Dashboard
- Login
- Registration
- Password Recovery

To ensure all app features work correctly, follow these additional steps:

1. Go to Settings > Permalinks and choose "Post name" as the permalink structure
2. Create a menu and assign it to the "Primary" location
3. Add Font Awesome to your site if not already included
4. Configure WooCommerce settings for checkout

== Customization ==

You can customize the theme colors in the WordPress Customizer under the "Theme Colors" section.

== Notes ==

This theme is designed specifically for BacklinkBoost applications and requires the Astra parent theme.
